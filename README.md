# CS161 — Exam 1: JavaScript Fundamentals

This is your starter repository for **CS161 Exam 1**.

## Getting Started

```bash
# Install Nibras CLI (if not already done)
npm install -g @nibras/cli

# Login with your GitHub account
nibras login

# View the task instructions
nibras task
```

## Working on the Exam

Edit `src/solution.js` to implement the three functions.

## Testing Locally

```bash
nibras test
```

## Submitting

```bash
nibras submit
```

Your submission will be verified automatically. You'll see a ✅ or ❌ result within 30 seconds.
